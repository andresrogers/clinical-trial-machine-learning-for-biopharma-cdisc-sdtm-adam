# Disease Signal Index

## Decision
The repository is locked to an Alzheimer’s dementia program narrative (`NTX-101`).

## Evidence from currently checked-in sources
- `data/external/cdisc_pilot/csr/narratives.txt` includes subjects described as having Alzheimer’s dementia.
- The same narrative file references placebo and xanomeline low dose treatment context.

## Why synthetic continuity is still required
Although the source backbone is standards-rich, it does not provide a clean single-program continuity arc from Phase I through Phase III for this portfolio storyline. A bounded synthetic continuity bridge is therefore required.

## Explicit non-claims

- It is not sponsor production or regulatory submission evidence.
